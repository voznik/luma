Loading Spinner

You can use the loading spinner whenever you're dynamically loading content.

<div class="spinner">
  <div class="pulse"></div>
  <div class="pulse"></div>
</div>

$color-spinner Color for the loading spinner color. Defaults to $color-primary.

$spinner-size Size of the loading spinner.
