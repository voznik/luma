Buttons

Buttons come in two basic styles: outlined and filled.

Outlined Buttons

Submit Primary Information Success Danger 
<div class="button-group">
  <a class="button is-outline">Submit</a>
  <a class="button is-outline is-primary">Primary</a>
  <a class="button is-outline is-info">Information</a>
  <a class="button is-outline is-success">Success</a>
  <a class="button is-outline is-danger">Danger</a>
</div>

Filled Buttons

Submit Primary Information Success Warning Danger 
<div class="button-group">
  <a class="button">Submit</a>
  <a class="button is-primary">Primary</a>
  <a class="button is-info">Information</a>
  <a class="button is-success">Success</a>
  <a class="button is-danger">Danger</a>
</div>

And with has-depth:

Submit Primary Information Success Warning Danger 
<div class="button-group">
  <a class="button has-depth">Submit</a>
  <a class="button has-depth is-primary">Primary</a>
  <a class="button has-depth is-info">Information</a>
  <a class="button has-depth is-success">Success</a>
  <a class="button has-depth is-danger">Danger</a>
</div>

Icons

Feel free to use any icon in conjunction with a button:

Watch Now Watch Now Delete Follow 
<!-- With Material Icons -->
<div class="button-group">
  <a class="button is-outline">
    <span class="icon">
      <i class="mdi mdi-play"></i>
    </span>
    <span>Watch Now>
  </a>
  <a class="button is-outline">
    <span>Watch Now>
    <span class="icon">
      <i class="mdi mdi-play"></i>
    </span>
  </a>
  <a class="button is-outline is-danger">
    <span>Delete>
    <span class="icon">
      <i class="mdi mdi-close"></i>
    </span>
  </a>
  <a class="button is-outline">
    <span>Follow>
    <span class="icon">
      <i class="mdi mdi-account-plus"></i>
    </span>
  </a>

There is also a special is-animated class that you can use:

Submit Follow Delete 
<!-- With Material Icons -->
<div class="button-group">
  <a class="button is-outline is-animated">
    <span>Submit>
    <span class="icon">
      <i class="mdi mdi-chevron-right"></i>
    </span>
  </a>
  <a class="button is-outline is-animated">
    <span>Follow>
    <span class="icon">
      <i class="mdi mdi-account-plus"></i>
    </span>
  </a>
  <a class="button is-danger has-depth is-animated">
    <span>Delete>
    <span class="icon">
      <i class="mdi mdi-close"></i>
    </span>
  </a>

$color-button Default font color for filled buttons when no is-* is specified.

$color-button-bg Default background color for filled buttons.

$color-button-outline-border Default border color for outline buttons when no is-* is specified.

$color-button-outline Default font color for outline buttons.
