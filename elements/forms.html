Forms

Why should forms be ugly? Luma provides custom styling for all common input elements.

<form>
  <div class="field">
    <label for="name">Name</label>
    <input type="text" id="name" name="name" placeholder="Your Name">
  </div>
  <div class="field">
    <label for="message">Message</label>
    <textarea id="message" name="messae" placeholder="Send me a message"></textarea>
  </div>
  <div class="field">
    <input type="checkbox" id="terms" name="terms">
    <label for="terms">I accept all the evil terms and conditions.</label>
  </div>
</form>

Form Elements

Text Inputs

<div class="field">
  <label for="title">Title</label>
  <input type="text" id="title" name="title" placeholder="Title">
</div>
<div class="field">
  <label for="title-disabled">Disabled</label>
  <input type="text" id="title-disabled" name="title-disabled" disabled>
</div>

If you include the Luma Javascript file, the textarea will automatically resize (shrink/expand) on user input.

Checkboxes

Luma uses custom CSS checkboxes. Check them out:

<div class="field">
  <input type="checkbox" id="check" name="check">
  <label for="check">Check 123</label>
</div>
<div class="field">
  <input type="checkbox" id="check-disabled" name="check-disabled" disabled>
  <label for="check-disabled">Disabled Checkbox</label>
</div>

Radios

<div class="field">

</div>
<div class="field">

</div>
<div class="field">

For the custom selects and file uploads the Luma Javascript is required.

$color-checkmark Color of the checkmark itself (white by default)

$color-checkbox Color of the background of a checked checkbox

$color-radio Marked radio background color

$color-select-bg Background color of the select list

$color-selected-bg Background color on select option hover
